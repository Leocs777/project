# TMS Labs Website

Static bilingual website for TMS Labs, the public brand of Trinity Management Services LLC.

## Pages

- English: `/`, `/real-estate/`, `/trade/`, `/apps/`, `/about/`, `/contact/`, `/privacy/`
- Chinese: `/zh/`, `/zh/real-estate/`, `/zh/trade/`, `/zh/apps/`, `/zh/about/`, `/zh/contact/`, `/zh/privacy/`

## Contact Routing

The contact form prepares an email to `hello@tmslabs.net`.

Snuggo Pet inquiries can also use `hello@snuggopet.com`.

## Brand Assets

- Full logo: `assets/images/tmslabs-logo.svg`
- Mark: `assets/images/tmslabs-mark.svg`
- Favicon: `favicon.svg`
- Brand notes: `BRAND.md`

## Deployment

This is a static site. Deploy the full folder contents to any static host, then point `tmslabs.net` to that host.

Recommended static hosts:

- Cloudflare Pages
- Netlify
- Vercel static deployment

## Before Launch

- Add a real Calendly link if scheduling should open directly.
- Replace app mockups with CardNest and Tripora screenshots when available.
- Replace or expand product visuals for Snuggo Pet and automotive accessories.
- Confirm any public company address or phone number before adding it.
